# Book History

**Book History** es una herramienta creada con el proposito de ayudar al personal de la biblioteca del _Colegio de Bachilleres de Estado de Jalisco_.

## Tecnologias usadas:
- Backend:
    - Express
    - TypeScript
    - MongoDB
    - NodeJS
    - Yarn
- Frontend:
    - ReactJS
    - TypeScript
    - TailwindCSS
    - Vite
    - NodeJS
    - Yarn
